# HandScanAI Frontend

## Environment Setup

1. Copy `.env.example` to `.env`.
2. Fill in the `VITE_API_BASE_URL` with the appropriate value.

## Installation

You can use `npm`, `yarn`, or `pnpm` to install the dependencies and run the development server.

### Using npm

```bash
npm install
npm run dev
```

### Using yarn

```bash
yarn install
yarn dev
```

### Using pnpm

```bash
pnpm install
pnpm dev
```
